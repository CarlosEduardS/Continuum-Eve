Este projeto integra robótica, engenharia e inteligência artificial para criar soluções inovadoras. Aqui você encontra uma visão geral e os dados principais do robô.

Sobre o Projeto

O PROJECT-EDEN combina conceitos de engenharia mecânica, robótica e eletrônica para desenvolver sistemas autônomos. Nosso objetivo é explorar como sensores e inteligência artificial podem melhorar a interação com o ambiente.
